## Installation

```bash
$ npm install
```

## Running Redis (with docker)

```bash
docker run --name redis -p 6379:6379 -d -t redis:alpine
```

## Running the app

```bash
# start publisher
$ npm run start:pub

# start subscriber
$ npm run start:sub
```
